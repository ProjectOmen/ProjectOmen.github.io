What Are Evolved Recipes?

Evolved recipes let players combine ingredients dynamically. Unlike regular craftRecipe blocks (which have fixed inputs and outputs), evolved recipes define a base item that can be transformed by adding ingredients to it.

Think sandwiches, soups, salads — any recipe where the player picks what to add from a pool of valid ingredients.

How It Works (Two Parts)

Part 1: The evolvedrecipe Block

This defines the recipe itself — what base item to start with, what the result is, and how many ingredients can be added.

Part 2: The EvolvedRecipe Property on Items

Each ingredient item needs an EvolvedRecipe property in its item script that says which evolved recipes it can be added to and at what weight. For example:

EvolvedRecipe = Sandwich:6;Burger:6,

This means the item can be added to any evolved recipe with Template:Sandwich (weight 6) or Template:Burger (weight 6).

Fields Explained

FieldDescription
Recipe NameThe identifier after evolvedrecipe (e.g. Sandwich)
BaseItemThe starting item that gets transformed (e.g. BreadSlices)
ResultItemThe item produced after adding ingredients (e.g. Sandwich)
NameDisplay name shown in the crafting menu (e.g. Make Sandwich)
MaxItemsMaximum number of ingredients that can be added
TemplateTemplate name that ingredient items reference in their EvolvedRecipe property. Multiple recipes can share the same template.
CookableIf true, the result can be cooked
AddIngredientIfCookedIf true, ingredients can be added even after the base/result is cooked
CanAddSpicesEmptyIf true, spices can be added before any other ingredients
AllowFrozenItemIf true, frozen ingredients can be used
IsHiddenIf true, hides the recipe from the crafting menu
AddIngredientSoundSound played when adding an ingredient (default: AddItemInRecipe)

Template System

The Template field is the glue between the recipe and its ingredients. When an item has EvolvedRecipe = Sandwich:6, it can be added to any evolved recipe where Template:Sandwich. This means two different recipes (e.g. Sandwich and Baguette) can share the same ingredient pool by using the same template name.

Output Format

evolvedrecipe Sandwich
{
    BaseItem:BreadSlices,
    MaxItems:4,
    ResultItem:Sandwich,
    Name:Make Sandwich,
    CanAddSpicesEmpty:true,
    AddIngredientIfCooked:true,
    Template:Sandwich,
}

Evolved recipe scripts go in your mod's media/scripts/ folder alongside other script files.

Ingredient Item Setup

For each ingredient that can be used in your evolved recipe, add the EvolvedRecipe property to the item's script:

item Tomato
{
    ...
    EvolvedRecipe = Sandwich:6;Burger:6,
    ...
}

The number after the colon is the "weight" — it affects how much hunger/nutrition the ingredient contributes. Use the ItemForge tool to build item scripts with the EvolvedRecipe property.
