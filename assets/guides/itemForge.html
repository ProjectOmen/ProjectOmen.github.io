Getting Started

ItemForge helps you build Project Zomboid Build 42 item script blocks without writing code by hand. Fill in a form, and it generates correctly formatted item scripts for your mod.

Quick Workflow

  1. Load an existing item script file (optional) — browse and edit existing items.
  2. Build your item — pick a Type, fill in the fields that appear. Different types show different property sections.
  3. Save to your session — click "Save Current Item" to add it to the list. Build multiple items per session.
  4. Export — copy or download your item(s). Translation lines for Items_EN.txt are auto-generated.

Item Types

PZ supports 14 item types. Each type exposes different properties:

  • Normal — Generic items (tools, parts, materials)
  • Weapon — Melee and ranged weapons (~65 properties)
  • Food — Edible items with nutrition, freshness, cooking (~40 properties)
  • Literature — Books, magazines, skill trainers
  • Drainable — Items with a use bar (lighters, medical supplies)
  • Clothing — Wearable items with defense, insulation
  • Container — Bags, boxes with capacity
  • WeaponPart — Attachments for weapons
  • Key — Keys and padlocks
  • Moveable — Furniture and placeable objects
  • Radio — Radios and communication devices
  • AlarmClock — Alarm clock items
  • AlarmClockClothing — Wearable alarm (like a watch)
  • Map — Map items

Common Fields (All Types)

FieldDescription
ItemIDUnique identifier with no spaces (e.g. MyCustomSword)
DisplayNameThe name players see in-game
TypeOne of the 14 item types listed above
WeightItem weight in kilograms
IconIcon texture name
DisplayCategoryCategory shown in inventory UI
TagsSemicolon-separated tags for recipe matching

Custom Properties (ModData)

Any property not recognized by the game engine is automatically written to the item's ModData. You can add custom key-value properties in the "Custom Properties" section. Access them in Lua with item:getModData().YourKey.

Translation Lines

Each item needs a translation entry in Items_EN.txt:

DisplayName_YourItemID = "Display Name Here",

Output Format

module Base
{
    item MyItem
    {
        DisplayName = My Custom Item,
        Type = Normal,
        Weight = 1.0,
        Icon = MyIcon,
        Tags = TagA;TagB,
    }
}

Item script files go in your mod's media/scripts/ folder. Translation entries go in media/lua/shared/Translate/EN/Items_EN.txt.
