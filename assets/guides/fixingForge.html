Getting Started

fixingForge helps you build Project Zomboid Build 42 fixing script blocks. Fixing blocks define how items can be repaired — what materials are needed, what skills are required, and how much condition is restored.

Quick Workflow

  1. Load an existing fixing script file (optional) — browse and edit existing fixing blocks.
  2. Build your fixing block — set the name, the item to repair, and add Fixer rows for each repair material.
  3. Save to your session — click "Save Current Fixing" to add it to the list.
  4. Export — copy or download your fixing block(s).

Fields Explained

FieldDescription
Fixing NameThe name of the fixing block (e.g. Fix Wood Axe). Can contain spaces.
RequireThe item ID to be repaired (e.g. WoodAxe).
GlobalItemAn additional item always needed for repair, consumed in the process. Format: Item=Qty
ConditionModifierMultiplier for how much condition is restored per repair (default 1.0). Lower = less repair per fix.

Fixer Rows

Each Fixer row represents one way to repair the item. The player only needs one of the listed fixers to perform the repair.

PartDescription
MaterialThe item used for repair (e.g. Woodglue, DuctTape, Nails)
QuantityHow many of the material are consumed. Leave empty for 1.
SkillOptional skill required for this repair method (e.g. Woodwork)
Skill LevelLevel required for the skill. Only used if Skill is set.

Available Skills

Axe, Blunt, SmallBlunt, LongBlade, SmallBlade, Spear, Maintenance, Aiming, Reloading, Woodwork, Cooking, Farming, Doctor, Electricity, MetalWelding, Mechanics, Tailoring, Fishing, Trapping, PlantScavenging, Fitness, Strength, Sprinting, Lightfoot, Nimble, Sneak

Output Format

fixing Fix Wood Axe
{
    Require : WoodAxe,

    Fixer : Woodglue=2; Woodwork=2,
    Fixer : DuctTape=2,
    Fixer : Glue=2,
    Fixer : Scotchtape=4,
}

Fixing script files go in your mod's media/scripts/ folder.
