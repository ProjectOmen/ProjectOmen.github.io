Overview

The fluid block defines a custom fluid for Build 42's fluid system. Fluids can be beverages, fuels, industrial liquids, or anything else that fills containers.

Basic Structure

fluid MyFluid
{
    Color           = 1.000 : 0.500 : 0.200,
    DisplayName     = FLUID_MY_FLUID,

    Categories
    {
        Beverage,
    }
}

Top-Level Fields

FieldDescription
ColorRGB values from 0 to 1, separated by colons. E.g. 1.000 : 0.894 : 0.769
ColorReferencePlaintext color name (e.g. Red). Alternative to RGB values.
DisplayNameTranslation key for the fluid name.

Categories Block

Lists categories this fluid belongs to. One per line, comma-separated.

Properties Block

Numeric properties that define what happens when the fluid is consumed.

Poison Block

FieldDescription
maxEffectSeverity: None, Low, Medium, Extreme, Deadly.
minAmountMinimum poison amount (0 to 1.0).
diluteRatioHow much poison dilutes when blended with other fluids.

Blend Lists

Control which fluids can or cannot be mixed together. Two approaches:

  • Script reference: BlendWhiteList = MyFilter, — references a separate filter script.
  • Inline block: Define allowed/blocked fluids and categories directly.
BlendWhiteList
{
    whitelist = true,
    fluids
    {
        Water,
    }
    categories
    {
        Beverage,
    }
}
