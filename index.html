Project Omen

Free, browser-based tools for Project Zomboid Build 42 modders. No installs, no frameworks — just open and build.

All tools run entirely in your browser. Nothing is uploaded.

craftRecipeForge Live

Visual craftRecipe builder for B42. Fill in a form, get correctly formatted recipe scripts. Supports inputs, outputs, itemMapper, overlayMapper, smart file insertion, and batch export.

Open craftRecipeForge →

itemForge Live

Item script builder for B42. Define items with all their properties — weight, type, tags, icon, categories — through a visual form. Supports all 14 item types, custom ModData properties, and batch export.

Open itemForge →

fixingForge Live

Fixing block builder. Define repair materials, skill requirements, and condition modifiers for any item. Dynamic fixer rows with searchable skill lists and batch export.

Open fixingForge →

evolvedRecipesForge Live

Evolved recipe builder for B42. Create sandwich, soup, stew, and other evolved recipes with base items, result items, templates, and all boolean flags.

Open evolvedRecipesForge →

fluidForge Live

Fluid script builder for B42. Define custom fluids with color, categories, nutrition properties, poison settings, and blend rules with inline or script-referenced filters.

Open fluidForge →

modScaffold Live

Mod folder scaffolder. Generate a correct B42 mod folder structure with mod.info, scripts, Lua, translations, and textures. Download as a ready-to-go ZIP.

Open modScaffold →

vehiclesForge Coming Soon

Vehicle script builder with full template support. Define vehicles, parts, passengers, physics, sounds, and skins — with the first comprehensive vehicle modding reference for B42.
